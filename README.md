# Secure Login Web Application

A Java Spring Boot web application demonstrating user authentication, login management, MVC architecture, and secure backend development concepts.

## Overview

This project implements a secure login system using Spring Boot and layered backend architecture principles.

The application demonstrates:
- User authentication
- Login form handling
- MVC application structure
- Backend/frontend integration
- Secure application workflow

## Features

- User login functionality
- Authentication workflow
- MVC architecture
- HTML template rendering
- Backend route handling
- Layered application structure

## Technologies Used

- Java
- Spring Boot
- Maven
- HTML
- Thymeleaf
- MVC Architecture

## How to Run

### Using Maven Wrapper

```bash
./mvnw spring-boot:run
```

### Or on Windows

```bash
mvnw.cmd spring-boot:run
```
